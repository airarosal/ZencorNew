# Zencor | Zeal Ecological Nature Corporation Website

Welcome to my web development project! This is a responsive website I built for **Zencor**, an agricultural business in Pampanga that helps Filipino farmers.

## рџ“– About the Project

For this activity, I created a 4-page website to showcase Zencor's products and history. My goal was to make it look like a real, professional business site while following all the SEO and design requirements for the subject.

**What I built:**
* **4 Main Pages:** Home, About, Products, and Contact.
* **Mobile Friendly:** The layout adjusts to phone screens (I used a hamburger menu for mobile).
* **Theme:** I moved away from basic colors and used a "Vintage/Rustic" theme (Cream, Dark Green, and Gold) to fit the farming vibe.
* **Real Data:** I used actual product names and prices from Zencor's inventory.
* **Map Integration:** I embedded a Google Map that pinpoints the exact office in Santa Ana, Pampanga.

## рџљЂ SEO & Coding Standards
I made sure to follow the checklist for On-Page SEO:
* **Meta Tags:** Added unique descriptions and keywords for every page.
* **Semantic HTML:** I used proper tags like `<nav>`, `<main>`, and `<footer>` instead of just using divs.
* **Images:** All images have `alt` text for accessibility.
* **Links:** The "Read Our Story" button links directly to a specific section on the About page.
